# fglm
Fortran GeneraL Math modules.

-----

### FAQ:

-----

Question: That is the most forced acronym I've ever seen.

Answer: Yes, I know.

-----

### What:

-----

A very simple math library for Fortran.

Has a few types:

- matrix4f
- vector2d
- vector2f
- vector2i
- vector3d
- vector3f
- vector3i

(Double, Float, Integer32)
